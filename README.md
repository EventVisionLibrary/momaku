# Setup

## Install glfw

```
sudo apt install libglfw3 libglfw3-dev
```

## Install requirements

```
pip install -r requirements.txt
```

# Usage

## Rendering environments

```
python env.py
```


## Training with DQN

```
cd src
python dqn_train_avoidance.py 
```

## Training with SNN

```
python train.py
```
